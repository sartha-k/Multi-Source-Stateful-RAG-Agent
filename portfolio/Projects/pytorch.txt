Autograd in PyTorch
This repository contains my hands-on implementation and exploration of PyTorch Autograd.

The goal of this repository is to deeply understand how automatic differentiation works before building full neural networks.

📌 Topics Covered
requires_grad=True
Computation graphs
grad_fn
backward()
Chain rule
Gradient accumulation
Detach and no_grad
Manual neuron example (forward + backward)
Each concept focuses on a core concept of automatic differentiation.

Understand how PyTorch builds computation graphs
Learn how gradients are calculated automatically
Build intuition for backpropagation
Prepare for neural network implementation
